# 如何设置中文字体的斜体？

## 用其它字体代替

一般使用其他字体（如楷体）代替斜体：

```typst
#let 西文字体 = "Libertinus Serif"
#show emph: text.with(font: (西文字体, "Kaiti"))
孔乙己_上大人_
```

## 另法：强行倾斜

伪斜体也可使用[`skew`函数](https://typst.app/docs/reference/layout/skew/)：
```typst
#skew(ax: -12deg)[111]111
```
但是针对一大段效果并不好。
